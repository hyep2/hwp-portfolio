Amiibo Locator


  • Amiibo Locator Link
  • Github Repo Link

Description: This Amiibo Locator page is devoted to helping gamers find all the information they want about amiibos, which are Nintendo collectable figures used to store data via NFC reader on certain Nintendo devices. User can select the game series they want to search the amiibos of, directing them to a new page with all the amiibo characters for that game. User will be presented with the image of each amiibo and the amiibo's game series, amiibo series, type, character name, and release dates from U.S.A, Europe, Australia, and Japan. This information has been fetched from the Amiibo API, coded by Nevin Vu. As an added bonus, a music player that will play all of the favorite tracks from Animal Crossing New Horizons is found in the home page. Lastly, on the retailers tab, we have conveniently linked some retailers where users can find the Amiibos that incite the urge to shop.


Weather Dashboard


  • Weather Dashboard Link
  • Github Repo Link

Description: This weather dashboard application allows users to search for the weather of the city they want. Upon selection of the city, the following information will display: temperature, wind speed, humidity, UV index, and weather icon for the current and the next five days. The application will also store the user's search log, by which the user can easily access weather information of previously searched cities.


Work Day Planner


  • Work Planner Link
  • Github Repo Link

Description: The work day scheduler is a simple calendar application that allows the user to add events for each work hour (9AM-5PM) of the day. When the user opens the scheduler on the browser, the current day is displayed at the top of the calendar. Then, below is the main section of the app where there are timeblocks for standard business hours that are each color-coded to indicate past, present, or future. Past is colored grey, present is colored red, and future is colored green. Also, the user can save the events if they press the 'save button', which will have the events persist even when the user refreshes the page.


Password Generator


  • Password Generator Link
  • Github Repo Link

Description: This password generator application will create a strong and unpredictable password for any user based on the selected criteria by user (number of characters, presence of lowercase, presence of uppercase, presence of numbers, and/or presence of special characters). It is often difficult for users to think of creative passwords that are easily leaked, especially if they incorporate personal information to their password. By applying this app, it will make it much more safe and convenient for users when creating an account.
